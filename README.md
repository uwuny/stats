# 📊 Анализатор реплеев

> Веб-инструмент для анализа файлов `.mtreplay` с визуализацией боёв и расширенной статистикой игроков.

⚙️ **Статус проекта:** в активной разработке  
🎮 **Назначение:** анализ реплеев, статистика игроков, визуализация событий боя

---

## 📌 О проекте

Анализатор реплеев предназначен для разбора `.mtreplay` файлов и визуального представления игровых событий.

Проект включает:
- парсер реплеев
- веб-интерфейс на GitHub Pages
- интерактивный анализатор боёв
- систему статистики игроков

---

## ✨ Возможности

### 🎥 Анализ реплеев
- Парсинг файлов `.mtreplay`
- Журнал нанесённого урона
- Просмотр состава команд
- Синхронизация событий боя с таймлайном

---

### 🗺️ Визуализация боя
- Интерактивная карта боя
- Отображение маршрутов техники
- Отображение уничтоженной техники
- Настраиваемые слои:
  - маршруты
  - выстрелы
  - урон
  - ники игроков

---

### 📊 Статистика игроков
- Нанесённый урон
- Полученный урон
- Попадания
- Ассист
- Выживаемость
- Средние значения статистики
- Оборудование *(в разработке)*

---

## 🌐 Веб-интерфейс (GitHub Pages)

Проект включает веб-интерфейс со статистикой и анализом боёв.

---

### 📑 Структура интерфейса

#### Вкладки:
- Статистика
- Оборудование *(в разработке)*

---

## 📊 Раздел «Статистика»

### Режимы отображения:
- 💥 Урон
- 🛡️ Получено урона
- 🎯 Попадания
- 🤝 Ассист
- ❤️ Выживаемость
- 📈 Среднее (переключатель отображения)

---

### ⏱️ Фильтрация

- Выбор события боя (с названием и временем)
- Фильтрация по датам (например: `01.06`, `04.06`)

---

### 📋 Таблица результатов

#### 🔗 Интерактивные элементы:

- Название клана противника  
  → переход на сайт Hemero со статистикой клана

- Название карты  
  → открытие HTML-страницы с интерактивной картой боя
